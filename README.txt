Funngro Demo - README
Files included:
 - index.html     (Company / Home page)
 - teen.html      (Teen Hub page)
 - seo_audit.txt  (Short sample SEO audit)
How to use:
 1) Replace 'YOURDOMAIN_OR_GITHUB.io' with your actual hosted URL.
 2) Optionally add a logo image (logo.png) in the same folder.
 3) Host on GitHub Pages / Netlify / Vercel by uploading these files.
 4) Test mobile view and then submit the hosted URL in the remark field.
