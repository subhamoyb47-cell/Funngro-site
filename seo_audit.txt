Funngro - Short SEO Audit (sample)
----------------------------------
1) Lighthouse summary (example):
   - Performance (mobile): 65
   - Accessibility: 85

2) Key issues found:
   - Large images slowing mobile load.
   - Missing image lazy-loading.
   - Some render-blocking CSS.

3) On-page SEO:
   - Title & meta description present on each page.
   - H1 present on both pages.
   - JSON-LD Organization included.

4) Recommendations (priority):

   - Inline critical CSS and defer non-critical CSS.
   - Add sitemap.xml and robots.txt.
   - Add descriptive alt text for any images.
   - Improve server response / use CDN if possible.
